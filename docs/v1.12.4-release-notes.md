# V1.12.4 Release Notes

## Knowledge Workspace Structure Refactor

V1.12.4 separates Knowledge Workspace state and browser-local document operations from the stable `KnowledgeWorkspace` composition entry.

- `useKnowledgeWorkspace` centralizes local document initialization, selection and filters, import, enable/disable, deletion, clear, and explicit backup-restore refresh actions.
- `KnowledgeBackupPanel` remains responsible for export, file selection, preview, merge, and replacement confirmation; it refreshes the workspace through its existing callback.
- `RagTestBench` remains a local Router/Retriever diagnostic and continues to write compact records through the existing RAG Test History storage module.

## Compatibility and Validation

- Knowledge Storage keys, backup JSON format, legacy migration, record filtering, and RAG API behavior are unchanged.
- Playwright adds a local Knowledge Workspace smoke flow for document creation, selection, enable/disable, refresh recovery, and deletion. Existing backup and Client Storage browser tests remain in place.
- V1.12.4 does not invoke a Real API; the V1.12.1 deployment smoke test remains the Real API baseline.
