# V2.2.3 Release Notes — 界面体验与产品质感优化

## 版本定位

V2.2.3 是企业工作台界面收口版本。它统一设计语言和页面状态，优化 Chat、知识库与企业知识包导入、评测中心、运行监控、业务工具、首页和 About，并补齐窄屏、键盘和 reduced-motion 验证。

## 设计审查

本版本实际使用项目级 `redesign-existing-projects` Skill 进行界面审查，但 Skill 只作为设计指导。企业内部 Agent 工作台的高信息密度、专业、克制和可信定位优先于通用“高级感”建议。

采用：视觉层级、间距/对齐、组件一致性、状态完整性、响应式、交互反馈和可访问性。

未采用：外部字体、营销式超大标题、巨大留白、破格不对称、玻璃拟态、复杂纹理、夸张渐变、悬浮卡片、视差、滚动揭示和大幅位移动画。

## 主要变化

- 全局 Token、当前页导航、Skip Link、统一焦点环、状态组件和 reduced-motion；
- Chat 改进会话层级、文档式长回答、RAG 依据、安全 Trace、Composer 和 390px 操作可达性；
- Knowledge 改进页面文案、文档选择、知识包、批量导入预览、中文进度和部分失败反馈；
- Evaluation 改进评测配置、核心指标、失败定位、表格语义、历史确认和报告反馈；
- Ops 改进访问口令语义、健康状态、存储模式、导入与错误分布中文化；
- Tools 改进流程分组、技术详情折叠和执行结果 live feedback；
- 首页改为紧凑推荐路径和真实能力边界；About 更新到 PostgreSQL 匿名工作区、企业知识包、上下文管理和当前安全边界；
- 增加 Loading、Error 和 404 页面，以及通用 UI/无障碍测试。

## 兼容性与边界

- 未修改业务核心逻辑；
- 未修改数据库 Schema，未新增 Prisma migration；
- 未修改 Router、RAG、工具调用、Real API、Streaming、Completion Fence、Context Manager、Rolling Summary、Summary Patch、CAS、Repository、Workspace Cookie 或 Import claim/lease；
- 未新增运行时 UI 依赖、外部字体或大型图表库；
- 未加入完整登录、权限、COS、OCR、长期记忆或新业务场景；
- Node 保持 20.19.5，Prisma / Prisma Client 保持 6.19.3，Undici 保持 7.28.0；
- 未修改 `.env.local`，未读取或输出 Secret；
- 未部署腾讯云，线上继续运行 V2.0.4。

完整 UI 规范见 `docs/v2.2.3-ui-guidelines.md`。
