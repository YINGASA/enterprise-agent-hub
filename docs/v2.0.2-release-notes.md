# Enterprise Agent Hub V2.0.2 Release Notes

## Scope

V2.0.2 adds streaming answers and generation controls without changing the Conversation storage contract, the bounded multi-turn context algorithm, or the existing JSON Agent API.

## Highlights

- Adds `POST /api/agent/stream` with typed NDJSON events for run start, safe pipeline phases, answer deltas, completion, errors, and aborts.
- Keeps `POST /api/agent` unchanged for Evaluation, older clients, and non-streaming callers.
- Consumes OpenAI-compatible Real SSE with split-chunk, multi-event, heartbeat, `[DONE]`, timeout, and abort handling.
- Never exposes partial structured JSON. Real structured output is validated before completion, and providers that reject streaming fall back to the existing complete response path with `streamFallback=true`.
- Streams deterministic Mock chunks for repeatable local tests without calling a Real API.
- Renders Assistant output incrementally, supports Stop Generation, and allows retry of only the current stopped or failed turn without duplicating its user message.
- Extends the existing request epoch, origin conversation ID, target existence, and AbortController checks to every stream event.
- Persists only completed Assistant messages. Pending, stopped, and failed output never enters future context.
- Records at most one privacy-safe Ops summary per run. Deltas, prompts, conversation context, raw upstream SSE, partial errors, and full answers are excluded.

## Compatibility and Boundaries

- V2.0.0 and V2.0.1 Conversation data remain readable, and V1.12.5 Chat History migration remains available.
- Sources, tools, Trace, context metadata, Feedback, Router, RAG, and tool selection continue to belong to the completed Assistant message.
- This release does not add regeneration of completed historical answers, answer branches, edited prompts, server-side conversations, automatic summaries, long-term memory, file upload, or cross-device sync.
- Local automation uses deterministic Mock mode and does not require or call a Real API.
- Deployment should include a controlled Real streaming smoke test, including one upstream that supports streaming and one known fallback path.

## Release Gate

- Unit tests pass at 27 files and 98/98 tests; Playwright passes 18/18 browser scenarios.
- Typecheck, Full Mock Evaluation, production build, and `git diff --check` pass.
- Full Mock Evaluation remains 80/80 with a 100% pass rate.
- Production audit has no high or critical findings. Two known moderate advisories remain in Next.js 16.2.10's transitive PostCSS; the suggested forced fix would downgrade Next and is not used.
