# V1.12.1 Release Notes

V1.12.1 completes RAG Test History migration to the shared client storage adapter while preserving the existing storage key and RAG Test Bench behavior. Client-side history now uses versioned envelopes, legacy migration, invalid-record filtering, bounded retention, and safe corruption recovery.

Quality baseline: 17 test files, 48 passing tests, and Full Mock evaluation at 80/80. Client knowledge and history remain browser-local; Ops uses lightweight local JSONL storage; in-memory rate limiting is single-instance only. Production audit has two documented moderate transitive findings and no high or critical findings.
