# V1.9 Release Candidate 能力摘要

V1.9 是 Enterprise Agent Hub 当前的候选发布版本，重点从演示型项目收口为更接近真实可用的企业知识库 Agent 应用。

## 当前产品链路

1. `/knowledge`：管理系统内置知识包和浏览器本地用户文档。
2. `/chat`：输入业务问题，查看回答来源、工具调用、运行模式、反馈入口和 Trace。
3. `/tools`：按业务流程查看自动化工具能力，并将模板问题带入 Chat。
4. `/evaluation`：运行 80 条 Mock 回归评测，并导出评测报告。
5. `/about`：查看项目定位、架构边界和当前版本能力摘要。

## V1.9 当前能力

- 配置模型服务后，Chat 默认使用 Real API 作为主运行时。
- 未配置模型服务或离线验证时，保留开发模拟模式。
- Real API 调用失败时明确标记 `real_error_fallback`，不把兜底回答伪装成真实模型成功。
- RAG 增加来源过滤、短问题澄清和回答质量打磨。
- 知识库支持文档启用/禁用、建议测试问题、质量诊断和用户文档持久化。
- 业务工具按订单、规则、工单、JD、客服回复等流程组织。
- full Mock 评测保持 80 条稳定回归。

## 当前边界

- 用户导入文档、反馈和运行历史保存在浏览器 `localStorage`。
- 默认业务数据为模拟数据，没有连接真实企业系统。
- 当前检索为本地 Hybrid RAG + Mock Embedding Adapter，不是生产级向量数据库。
- Real API 凭据仅通过服务端环境变量配置，不应出现在 UI、代码、README 或文档中。
